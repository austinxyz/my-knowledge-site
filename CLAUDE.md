# CLAUDE.md — 硅谷华人理财顾问网站

Austin Xu 个人理财顾问落地页。基于 Claude Design 设计稿 `硅谷华人理财顾问-1c.dc.html` 实现。

## 品牌定位与目标受众

**定位：** 把理财当系统来优化。工程师逻辑 + CFP® 专业，客观按需、不推销、不制造焦虑。

**目标受众：** 30–45 岁湾区华人工程师家庭。典型画像：
- RSU 已开始 vest，股票占资产比例升高
- 对税务困惑，不确定有没有多交税
- 信任专业、反感推销，想要客观按需建议

**核心价值主张：** 帮工程师把每个节税与增长机会都利用起来，用熟悉的逻辑讲清复杂财务问题。

## 配色方案

| 主色（深藏青） | `#0f2740` | nav、hero 卡片、关于我区块、CTA 区块、深色按钮、标题文字 |
| 辅色（蓝） | `#4a86c4` | 强调、序号、链接、主按钮、mono 标签 |

**文字色：**
- 正文深：`#3f4c58` / `#4e5b68` / `#5c6773`
- 深色底浅字：`#a9bccf` / `#8aa4bd` / `#b0c2d4`
- 静默/说明：`#8494a2`

## 字体规范

- **中文 / 主字体：** `Noto Sans SC`（Google Fonts，weights 400/500/700/900），fallback `system-ui, sans-serif`
- **等宽 / 标签：** `ui-monospace, Menlo, monospace` — 用于序号（01/02/03）、英文小标签（`401K · ROTH`、`TAX PLANNING`）、`FOR SILICON VALLEY ENGINEERS`
- **字重用法：** h1/h2/统计数字 `900`；小标题/按钮 `600–700`；正文 `400–500`
- **字号锚点：** h1 46px / h2 30px / h3 18–19px / 正文 15–16.5px / 标签 12–13px

## 语气与 Tone

- 专业、克制、理性。工程师对工程师说话。
- 反推销、反焦虑。强调"客观、按需、可跟踪、可执行"。
- 用具体数字和框架代替情绪化措辞（"算清数字""输出路线图""提前一年规划"）。
- 中英混排：专业术语保留英文（RSU / 401K / Roth / Mega Backdoor / ISO / CFP®）。
- 免责声明必备：内容仅供教育参考，不构成投资、税务或法律建议。

1. **NAV** — sticky 顶部导航（主色底）。logo mark + "Austin Xu · Financial Advisor"，锚点 01 痛点 / 02 关于 / 03 套餐 + 预约按钮

7. **FOOTER** — 版权 + 免责声明（`#0b1e33` 底）
