Small rounded pill for eyebrow context, credentials, and "最受欢迎"-style markers.

```jsx
<Badge>RSU 已开始 vest？该做规划了</Badge>
<Badge variant="onAccent">最受欢迎</Badge>
<Badge mono>CFP® · 占位</Badge>
```

Variants: `tint` (default), `solid`, `onAccent` (for dark bands), `ghost`. Set `mono` for credential/placeholder chips.
