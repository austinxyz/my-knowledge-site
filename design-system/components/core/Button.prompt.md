Filled/outline action button in the brand accent; renders an anchor when `href` is set. Use `primary` for the main CTA, `secondary` for the adjacent alternative, `onAccent` inside dark accent bands.

```jsx
<Button href="#book">预约 15 分钟免费聊聊</Button>
<Button variant="secondary" href="#book">加微信咨询</Button>
<Button variant="onAccent" href="#">通过 Calendly 预约 →</Button>
<Button variant="primary" size="block">预约年度规划</Button>
```

Variants: `primary` (filled), `secondary` (outline), `onAccent` (white). Sizes: `md`, `sm`, `block`. Hover dims to 90% brightness.
